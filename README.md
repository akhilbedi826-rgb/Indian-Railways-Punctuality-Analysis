# 🚆 Indian Railways Punctuality Analysis

A complete Data Science project analyzing punctuality patterns across Indian Railways using Python, SQL concepts, Statistics, and Power BI.

---

## 📌 Project Overview

Indian Railways carries millions of passengers every day, making punctuality one of the most important operational metrics.

This project analyzes more than **55,000 railway records** to identify:

- Delay patterns
- Regional differences
- Seasonal effects
- Peak-hour congestion
- Train type performance
- Route efficiency
- Statistical significance of delay patterns

The project follows a complete Data Science workflow from data cleaning to business insights.

# 💡 Major Insights

✔ Northern railway zones experience significantly higher delays.

✔ Monsoon season produces the highest average delays.

✔ Premium trains like Vande Bharat and Rajdhani are considerably more punctual.

✔ Peak-hour operations increase delays substantially.

✔ Distance alone has very little impact on delays.

✔ Operational and infrastructure factors contribute more to delays than schedule length.
